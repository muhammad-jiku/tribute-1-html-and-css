# Tribute to Aristotle Onassis

## Project Description

This is a simple, responsive tribute web page dedicated to Aristotle Onassis, one of the most influential shipping magnates of the 20th century. The project demonstrates clean HTML and CSS practices, with a focus on accessibility and device responsiveness.

## Tech Stack

- HTML5
- CSS3 (responsive design, flexbox, gradients)

## About

The web page features:

- A brief biography and timeline of Aristotle Onassis
- Responsive layout for mobile, tablet, and desktop
- Modern, accessible color scheme and typography
- Deployed easily via GitHub Pages
